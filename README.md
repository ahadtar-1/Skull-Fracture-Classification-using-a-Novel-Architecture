# Skull-Fracture-Classification-using-a-Novel-Architecture
This research project implements a novel architecture for the purpose of classifying fracture CT scan and healthy CT scan images. Classification was implemented on an imbalanced, small, and complex dataset that contained a variety of fractures. 

The complete dataset cannot be released due to privacy constraints. Some images, however, are available for testing. Some grad-cam++ heatmaps are also available for observation. Each type of fracture present in the dataset is present in the heatmaps being recognized.
